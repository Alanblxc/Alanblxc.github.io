美化 windows 终端

最近重装了系统，windows 的美化又得重新弄了

所以顺便记录一下过程

先放几张成品图片



怎么样，好看吧

接下来分为个步骤

  1. 安装新版 pwsh

  2. 配置 terminal

  3. 安装 oh-my-posh

  4. 再安装其它几个模块

    第一步 安装 pwsh

    打开我们的微软商店

    搜索 pwsh


    就这个，安装

    第二步 配置 terminal

打开 windows terminal 设置的 json 文件

然后下载字体

下载这个 DejaVuSansMono NF

不然等会装 oh-my-posh 会乱码

安装 oh-my-posh

输入 set-executionpolicy remotesigned

允许脚本执行

安装命令

Install-Module git-aliases -AllowClobber -Scope CurrentUser > Install-Module posh-git -Scope CurrentUser > Install-Module oh-my-posh -Scope CurrentUser > Install-Module DirColors -Scope CurrentUser

Install-Module -Name Terminal-Icons -Repository PSGallery

等待安装，有问你选什么就输入 A

安装完后输入

notepad $PROFILE

然后在里面填入

Import-Module DirColors > Import-Module posh-git > Import-Module -Name Terminal-Icons > Import-Module git-aliases -DisableNameChecking > Import-Module oh-my-posh > Set-PoshPrompt -Theme cloud-native-azure > Set-PSReadLineOption -PredictionSource History > Set-PSReadlineKeyHandler -Key Tab -Function MenuComplete > Set-PSReadLineKeyHandler -Chord "Ctrl+f" -Function ForwardWord

然后就大功告成啦

有自动补全/历史记录等
